Important FAQ

What is the Document Object Model(Dom)?

A programming interface which is used to access and manipulate HTML documents using scripts or make dyynamic update content

How to select an element form the DOM?

There are multiple ways to select element form the dom among them mostly used are getElementById(),getElementsByClassName(),querySelector() etc. depending on the use there are very useful.

What is event delegation in the context of the DOM, and why is it useful?

Adding single event listener to parent element to handle events form its child, to reduce the number of event listener

How do you manipulate an element's attributes and styles using the DOM

There are multiple ways to select element form the dom among them mostly used are add(),remove(),removeAttribute(),setAttribute() etc. depending on the use there are very useful.
